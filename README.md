## About This Project

YelpCamp is a web application that allows users to:
- Browse and review campgrounds
- Create, edit, and delete campground listings
- Upload images
- User authentication and authorization
- Interactive features with secure data handling

## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: Passport.js
- **View Engine**: EJS
- **Image Storage**: Cloudinary
- **Security**: Sanitization, Session Management

## Quick Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example` or README_GENERATOR_GUIDE.md)
4. Start the server: `npm start`
5. Visit http://localhost:3000
